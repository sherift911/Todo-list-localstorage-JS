# 📝 To-Do List App (JavaScript)

A simple and interactive To-Do List application built using pure JavaScript.

👉 [View Project](https://sherift911.github.io/Todo-list-localstorage-JS/)
![Todo-list-localstorage-JS](images/site.png)

## 🚀 Features
- Add new tasks
- Delete individual tasks
- Delete all tasks
- Save tasks using Local Storage
- Tasks persist after page reload

## 🧠 How It Works
- Tasks are stored in an array
- Each task has:
  - id (unique باستخدام Date.now)
  - title
  - completed status
- البيانات بتتخزن في Local Storage باستخدام JSON

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript (Vanilla JS)
- Local Storage API

## 📂 Project Structure
  index.html
  style.css
  main.js

## ▶️ Usage
1. Enter a task in the input field
2. Click "Add Task"
3. Click "Delete" to remove a task
4. Click "Delete All" to clear all tasks

## 💾 Local Storage
All tasks are saved in browser local storage, so they remain after refreshing the page.
